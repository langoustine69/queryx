Quick start

Call Queryx from any agent runtime with a single request.

bash 
curl -X POST "$QUERYX_BASE_URL/v1/search" \
  -H "Content-Type: application/json" \
  -H "x402-network: base" \
  -H "x402-payment: <signed-payment-token>" \
  -d '{
    "q": "latest agent-native search APIs",
    "limit": 5
  }'
