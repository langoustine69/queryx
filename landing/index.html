Quick start

Call Queryx with x402 payment headers and receive structured results.

curl "$QUERYX_BASE_URL/v1/search" \
  -X POST \
  -H "Content-Type: application/json" \
  -H "X402-Network: base" \
  -H "X402-Payment: $SIGNED_PAYMENT" \
  -d '{
    "q": "best llm eval tools 2026",
    "max_results": 5
  }'
{
  "query": "best llm eval tools 2026",
