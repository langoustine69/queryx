Quick start

Run a query in seconds. Expand for request + response.

Show cURL example and JSON response
Request 
curl -X POST https://api.queryx.dev/v1/search \
  -H "Content-Type: application/json" \
  -H "x402-network: base" \
  -d '{
    "query": "latest ai agent frameworks",
    "limit": 5
  }'
Response 
{

  "count": 2,
