Quick start

Send a query and receive structured JSON immediately.

curl -X POST "https://api.queryx.ai/v1/search" \
  -H "content-type: application/json" \
  -H "x402-payment: <signed_payment_on_base>" \
  -d '{
    "query": "latest L2 throughput benchmarks",
    "top_k": 5
  }'
{
  "query": "latest L2 throughput benchmarks",
  "results": [
    {
      "title": "Benchmark roundup",
      "url": "https://example.com/benchmark-roundup",
      "snippet": "A comparative report on rollup performance...",
      "source": "web",
      "published_at": "2026-02-10T14:03:00Z"
    }
  ],
  "total_results": 1
}
