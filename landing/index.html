Quick start

One request in, structured JSON out.

cURL request + JSON response

Request

curl -X POST "$QUERYX_BASE_URL/v1/search" \
  -H "Content-Type: application/json" \
  -H "x402-payment: $SIGNED_PAYMENT_ON_BASE" \
  -d '{
    "query": "latest AI agent research",
    "num_results": 5
  }'

Response

{
  "query": "latest AI agent research",
  "cost_usd": 0.001,
  "network": "base",
  "results": [
    {
      "title": "Example result",
      "snippet": "Concise summary text...",
      "source": "news",
      "published_at": "2026-03-04T10:12:00Z"
    }
  ]
}
