Quick start

Run a single paid request and get JSON back.

cURL 
curl -X POST "https://api.queryx.dev/v1/search" \
  -H "Content-Type: application/json" \
  -H "X402-Network: base" \
  -d '{
    "q": "latest advances in battery recycling",
    "limit": 5
  }'
