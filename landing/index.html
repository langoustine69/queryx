Quick start

Make a request with a single curl command. Payment is handled by x402 on Base. 

curl -X POST https://api.queryx.dev/v1/search \
  -H "Content-Type: application/json" \
  -H "x402-network: base" \
  -d '{
    "query": "latest AI agent frameworks",
    "limit": 5
  }'
