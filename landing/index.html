Quick start

Run one request and get normalized JSON back.

cURL

curl -X POST https://api.queryx.dev/v1/search \
  -H "Content-Type: application/json" \
  -H "X402-Payment: <signed_payment_payload>" \
  -d '{
    "query": "latest base ecosystem updates",
    "limit": 5
  }'

JSON response

{
  "query": "latest base ecosystem updates",
  "results": [
    {
      "title": "Base ecosystem roundup",
      "url": "https://example.com/base-roundup",
      "snippet": "A summary of major launches and milestones..."
    }
  ],
  "usage": {
    "endpoint": "/v1/search",
    "cost_usd": 0.001
  }
}

How it works

1

Send query

Your agent calls a Queryx endpoint with a search prompt.

2

x402 payment on Base

Payment is attached to the request using x402 over Base rails.

3

Get structured JSON

Receive normalized results ready for tool use and chaining.
