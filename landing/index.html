Quick start

One call, one payment proof, one JSON response.

curl 
curl -X POST https://api.queryx.io/v1/search \
  -H "Content-Type: application/json" \
  -H "x402-payment: <base-payment-proof>" \
  -d '{"query":"latest ethereum L2 news","limit":5}'
